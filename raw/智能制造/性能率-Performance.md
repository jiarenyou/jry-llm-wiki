# 性能率 — Performance

## 定义

性能率衡量的是**设备在运行时间内，实际产出速度与理论最大速度之间的比值**。它回答：设备虽然开着了，但干得够快吗？

## 计算公式

$$性能率 = \frac{实际产量 \times 理论节拍时间}{实际运行时间} \times 100\%$$

等价形式：

$$性能率 = \frac{实际产量}{理论产量} \times 100\%$$

其中理论产量 = 实际运行时间 / [[节拍时间-Takt Time|理论节拍时间]]

## 速度损失的来源

| 损失类型 | 示例 | 特征 |
|----------|------|------|
| **空转与短暂停机** | 传感器误触发、卡料、小故障 | 单次 < 5 分钟，高频发生 |
| **降低运行速度** | 设备老化、参数保守设置、原料不适配 | 持续低速运行 |
| **不稳定运行** | 速度波动、加减速频繁 | 数据上体现为节拍方差大 |

## 为什么性能率最难抓？

性能率的"隐形损失"最大——操作工往往**不会报告短暂停机和轻微降速**，因为：
1. 每次损失很小，不值得报告
2. 设备没有完全停，只是变慢了
3. 但积少成多，可能吃掉 10-20% 的产能

**这就是数据采集的价值所在**：SCADA 可以逐秒采集设备速度，让隐性损失显性化。

## 数据采集要点

```
设备ID, 时间戳, 当前运行速度(件/分钟),
理论节拍时间, 累计产量, 空转标志位,
速度偏差百分比
```

## 基准值

| 等级 | 性能率 |
|------|--------|
| 世界级 | ≥ 95% |
| 行业平均 | ~85% |
| 需改善 | < 75% |

## 在 [[OEE-设备综合效率|OEE]] 中的角色

性能率是 [[OEE-设备综合效率|OEE]] 的第二个乘数因子。当 [[可用率-Availability|可用率]] 已经优化到位后，性能率是进一步提升 OEE 的关键战场。
