# 可用率 — Availability

## 定义

可用率衡量的是**设备在计划生产时间内，实际在运行的时间比例**。它回答一个简单的问题：设备该干活的时候，有没有在干活？

## 计算公式

$$可用率 = \frac{实际运行时间}{计划生产时间} \times 100\%$$

其中：
- **计划生产时间** = 总时间 - 计划停机时间（如休息、培训、无订单）
- **实际运行时间** = 计划生产时间 - 非计划停机时间

## 停机损失的来源

| 损失类型 | 示例 | 数据来源 |
|----------|------|----------|
| **设备故障** | 电机烧毁、液压泄漏 | SCADA 报警记录 |
| **设置与调整** | 换模、换线、参数调试 | MES 工单切换记录 |
| **待料停机** | 上游缺料、物流跟不上 | MES 物料拉动数据 |
| **待人员** | 操作工未到岗 | MES 人员排班 |

## 影响因素与关联指标

可用率直接受两个维护指标影响：

- [[MTBF-平均故障间隔时间|MTBF]] ↑ → 故障频率降低 → 可用率 ↑
- [[MTTR-平均修复时间|MTTR]] ↓ → 故障恢复更快 → 可用率 ↑

## 数据采集要点

从 SCADA/MES 中需要采集的关键字段：

```
设备ID, 时间戳, 设备状态(运行/停机/待机), 
停机原因代码, 停机开始时间, 停机结束时间,
计划生产开始时间, 计划生产结束时间
```

## 基准值

| 等级 | 可用率 |
|------|--------|
| 世界级 | ≥ 90% |
| 行业平均 | ~80% |
| 需改善 | < 70% |

## 在 [[OEE-设备综合效率|OEE]] 中的角色

可用率是 [[OEE-设备综合效率|OEE]] 三个乘数因子中的第一个，直接决定了设备的"时间底座"。可用率低意味着设备连开都没开起来，后续的性能和质量优化都是空谈。

## 关联指标

- [[OEE-设备综合效率|OEE]] — 可用率是其三要素之一
- [[MTBF-平均故障间隔时间|MTBF]] — 故障越少，可用率越高
- [[MTTR-平均修复时间|MTTR]] — 修复越快，可用率越高
- [[性能率-Performance|性能率]] — OEE 第二要素
- [[质量率-Quality|质量率]] — OEE 第三要素
