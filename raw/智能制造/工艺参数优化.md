# 工艺参数优化

## 定义

工艺参数优化是指通过系统化的方法，找到最优的工艺参数组合（温度、压力、速度、时间等），使得生产过程在满足质量要求的前提下，实现良率最大化、成本最小化、效率最高化。

## 方法演进

| 阶段 | 方法 | 特点 | 适用场景 |
|------|------|------|----------|
| **经验调参** | 老师傅经验 + 试错 | 依赖个人、不可复制 | 传统工厂 |
| **DOE试验设计** | 正交试验、全因子试验 | 科学化、但试验成本高 | 新产品导入 |
| **响应面法** | RSM拟合二次曲面 | 找到连续空间的最优解 | 参数较少的场景 |
| **贝叶斯优化** | 高斯过程+采集函数 | 少样本高效探索 | 昂贵实验优化 |
| **强化学习** | 智能体与环境交互 | 在线实时优化 | 连续生产过程 |

## 数据需求

### 输入（工艺参数）
- **温度**：加热温度、模具温度、环境温度
- **压力**：注射压力、保压压力、压缩空气压力
- **速度**：主轴转速、进给速度、线速度
- **时间**：保压时间、冷却时间、固化时间

### 输出（质量结果）
- 尺寸精度、表面粗糙度、外观缺陷、力学性能
- 关联指标：[[良率-Yield]]、[[FPY-直通率]]、[[CPK-过程能力指数]]

### 数据来源
- [[MES-制造执行系统]]：工序参数记录、质检结果
- [[SCADA-数据采集与监视]]：设备实时参数
- [[QMS-质量管理系统]]：质量检测数据

## 核心挑战

### 1. 多目标优化
通常不是优化单一目标，而是需要同时考虑：
- 良率最大化
- 成本最小化
- [[节拍时间-Takt Time|节拍时间]]最小化
- 参数必须在设备安全范围内

→ 需要用帕累托最优（Pareto Optimal）来权衡。

### 2. 参数耦合
工艺参数之间往往存在强耦合关系：
- 温度升高 → 流动性改善 → 但可能产生变形
- 压力增大 → 密实度提高 → 但可能导致飞边

→ 不能逐个参数独立优化，需要联合优化。

### 3. 工艺约束
- 设备能力约束（温度上限、压力上限）
- 安全约束（不能出现危险工况）
- 材料约束（温度窗口有限）

## 实施路径（三步走）

### 第一步：统计相关性分析
```
收集历史数据 → 计算参数与质量的相关系数 → 识别关键参数
```
- 工具：相关性矩阵、散点图矩阵、特征重要性排序
- 产出：识别出对质量影响最大的 3-5 个关键参数

### 第二步：建模与优化
```
关键参数 → 构建代理模型 → 优化搜索最优参数组合
```
- 代理模型：多项式回归、随机森林、高斯过程
- 优化搜索：网格搜索、贝叶斯优化、遗传算法
- 产出：推荐的最优参数组合及预期效果

### 第三步：在线调参与闭环
```
最优参数 → 下发到设备 → 实时监控效果 → 模型自更新
```
- 通过 [[MES-制造执行系统]] 或 [[SCADA-数据采集与监视]] 下发参数
- 用 [[SPC-统计过程控制]] 监控效果
- 模型持续学习，形成闭环

## 大数据工程师的角色

| 环节 | 数据工作 |
|------|----------|
| 数据准备 | 从 MES/SCADA 抽取工艺参数+质量结果，构建训练数据集 |
| 特征工程 | 参数衍生特征（均值、方差、变化率、累积量） |
| 数据质量 | 处理缺失值、异常值、时间对齐 |
| 模型部署 | 将优化模型封装为 API，集成到 [[实时数据流水线]] |
| 效果监控 | [[SPC-统计过程控制]] 看板，监控优化后的稳定性 |
