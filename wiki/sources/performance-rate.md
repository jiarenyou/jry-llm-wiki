## Summary
性能率是 OEE 三要素之一，衡量设备在运行时间内实际产出速度与理论最大速度之间的比值。它回答"设备虽然开着了，但干得够快吗"的问题。性能率的"隐形损失"最大，操作工往往不报告短暂停机和轻微降速，积少成多可能吃掉 10-20% 的产能。世界级基准为 >= 95%，行业平均约 85%。

## Key Claims
- 计算公式：性能率 = (实际产量 x 理论节拍时间) / 实际运行时间 x 100%
- 速度损失来源包括空转与短暂停机（单次 < 5 分钟但高频发生）、降低运行速度、不稳定运行
- 性能率最难抓的原因是"隐形损失"——短暂停机和轻微降速不会被操作工主动报告
- SCADA 可以逐秒采集设备速度，让隐性损失显性化，这就是数据采集的价值所在
- 性能率的计算依赖理论节拍时间，与节拍时间指标紧密关联
